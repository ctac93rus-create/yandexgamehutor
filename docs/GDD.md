## Milestone 1: Merge Loop
- Игрок работает на доске 6x7: перемещает предметы, совмещает одинаковые tier-предметы, раскрывает следующую стадию цепочки.
- Генератор (например костёр) тратит заряды на выдачу базового ресурса, заряды восстанавливаются по таймеру.
- При заполненной доске применяется overflow-политика: выдача безопасной компенсации в пыли, чтобы не возникал софтлок.
- UI: верхняя панель (золото/пыль), нижняя навигация (меню/хутор/рейд), тосты для обратной связи.
