# Mobile smoke checklist (pre-publish)

- [ ] Запуск игры на мобилке: корректный старт, safe-area, текст не упирается в края.
- [ ] Читаемость главного меню: все кнопки/иконки видны и нажимаются.
- [ ] Tutorial: проходит по шагам; после reset tutorial снова появляется.
- [ ] Raid: победа и поражение корректно завершаются.
- [ ] Raid reward: модалка показывает kills/score/reward/multipliers, claim работает.
- [ ] Raid rewarded x2: реклама успешна -> награда удваивается; отмена не ломает flow.
- [ ] Hut upgrades: цена в UI совпадает со списанием ресурсов после покупки.
- [ ] Story/Daily claims: награды начисляются корректно.
- [ ] Leaderboards:
  - [ ] локально показывается unavailable,
  - [ ] на платформе при `auth_required` есть вход,
  - [ ] refresh обновляет данные.
- [ ] Cloud save: синк между сессиями, конфликт `updatedAt` разрешается корректно.
- [ ] Ads:
  - [ ] interstitial уважает min interval,
  - [ ] rewarded уважает cooldown,
  - [ ] `disableAds` отключает рекламу.
- [ ] Локализация: RU/EN строки в меню/прогрессе/рейде/настройках.
- [ ] Финальная сборка: `npm run zip` создаёт архив без ошибок.
