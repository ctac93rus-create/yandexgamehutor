# LIVEOPS

## Прогрессия (Milestone 3)
- Сюжетные квесты разделены по главам `chapter_1..chapter_4`.
- Базово открыт `chapter_1`, следующие главы открываются:
  - автоматически после завершения предыдущей главы,
  - или через апгрейды хутора в `economy.json`.
- События прогрессии: `merge_done`, `generator_spawn`, `raid_complete`, `raid_win`, `hut_upgrade`.

## Ежедневные квесты
- 3 слота на день.
- Обновление происходит 1 раз в UTC-день.
- Ролл квестов детерминированный, чтобы исключить рассинхрон между сессиями в пределах одного дня.

## Контентные рекомендации
- Добавлять новые ежедневки можно в `quests_daily.json` без изменений кода, если используются существующие event-keys.
- Для новых глав добавлять `chapter_X` в `quests_story.json` и при необходимости upgrade в `economy.json`.
- Для событий в новых режимах расширять `QuestEngine.onEvent` и публиковать события в нужной сцене.
